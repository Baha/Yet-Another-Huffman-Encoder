Este es el directorio que contiene el proyecto para la segunda
practica de COD de Adrian Palacios Corella.

La opcion que escogio el autor fue la tercera (completa), por lo
que se ha tratado de configurar unas cuantas pruebas para probar
los cuatro modulos por separado. Para una breve explicacion del
proyecto, puede consultarse el ficher pdf en la carpeta "guide".

Se ha creado un make que puede usarse del siguiente modo:

baha@30L:~/Yet-Another-Huffman-Encoder$ make ex1

Esto creara un fichero binario llamado "modulo1" que puede 
usarse del siguiente modo:

baha@30L:~/Yet-Another-Huffman-Encoder$ ./modulo1 fichero.txt

Desde el modulo 1 hasta el modulo 3, solo se ha de pasar el nombre
de un fichero por la linea de comandos.

El modulo 4 necesita de 2 ficheros para ejecutarse (el primero
es el archivo codificado i el segundo es el archivo destino).

baha@30L:~/Yet-Another-Huffman-Encoder$ ./modulo1 fichero.huf fichero.txt
